# WhatsApp-PWA

A modern **WhatsApp-inspired Progressive Web App (PWA)** built with a focus on **UI/UX design** and **Liquid Glass aesthetics**, targeting cross-platform compatibility using a simple Node.js + Express backend.

---

## 🚀 Project Goals

- Replicate a **WhatsApp-style chat interface**
- Implement a **Liquid Glass-inspired UI system**
- Ensure **cross-platform compatibility** (Chrome-first approach)
- Serve as a **foundation for a full messaging app**

---

## 🧊 Features (Current)

- Chat list UI (dark mode, WhatsApp-inspired)
- Search bar and filter pills
- Header with action buttons
- Floating **Liquid Glass bottom navigation**
- Responsive mobile-first layout
- Lightweight Node.js + Express server

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, Vanilla JavaScript  
- **Backend:** Node.js + Express  
- **Architecture:** Static frontend served via Express  
- **Design Approach:** Glassmorphism + iOS-inspired depth and motion
